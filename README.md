# CIRCL PE32 certificate dump utility 

## Description
A tool to dump (and parse) an embedded certificate of a windows binary

## Requirements
* pedump (http://pedump.me/ - gem install pedump)
* bc (http://www.gnu.org/software/bc/)
* grep, tr, cut, dd, openssl
